Analysis of motivational speeches

An adaptation of the script on topic modeling by Alice Zhao, this time on motivational speeches by 10 motivational speakers.

Analyses done:
- data cleaning
- tokenizing
- pos tagging (NLTK)
- finding common and unique words
- visualizing (WordCloud and Matplotlib)
- sentiment analysis (TextBlob)
- topic modeling (LDA)
- text generation (Markov chain)
